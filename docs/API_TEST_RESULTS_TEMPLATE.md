## 테스트 실행 방법

### 1. 백엔드 서버 실행

```bash
cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### 2. 테스트 실행

```bash
cd backend
python test_all_apis.py
```

### 3. 결과 확인

테스트 결과는 다음 위치에 저장됩니다:

- JSON: `backend/test_results/api_test_results_YYYYMMDD_HHMMSS.json`
- 마크다운 보고서 생성: `python backend/generate_test_report.py <json_file> docs/API_TEST_RESULTS.md`

---

**참고**: 이 문서는 템플릿입니다. 실제 테스트를 실행한 후 `generate_test_report.py` 스크립트를 사용하여 업데이트하세요.
